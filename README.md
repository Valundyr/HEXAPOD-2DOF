# HEXAPOD-2DOF
Autonomous 12-DOF Hexapod robot powered by Raspberry Pi Pico W. Features a real-time Web UI for WiFi control, custom C++ Inverse Kinematics (Tripod, Centipede, &amp; Sideways gaits), active MPU6050 gyro-stabilization, and VL53L0X Lidar for obstacle avoidance. Implements a robust State Machine for smooth locomotion.
